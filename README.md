# Xeno Retail Dashboard & Expansion

This repository contains the interactive prototype and documentation for the Xeno Product Manager Internship Assignment.

## Live Prototype
The application is built with Next.js, Tailwind CSS, and shadcn/ui.
To run it locally:
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open [http://localhost:3000](http://localhost:3000).

## Project Structure
- **/app**: Contains the Next.js App Router pages (`dashboard`, `campaigns`, `audience`, `loyalty`, `analytics`).
- **/components**: Reusable UI components (KPI cards, charts, tables, dialogs).
- **/Relevent Files**: Contains the **Final Submission Document** and **Video Script**.
- **artifacts**: Contains PM thinking, wireframes, and design specs.

## Assignment Overview
This dashboard addresses the "10-Second Rule," helping retail marketers immediately understand account health and take high-value actions upon login.

**Key Features:**
- **Command Center Dashboard:** Revenue vs. Spend visualization.
- **Unified Campaigns:** Multi-channel management (WhatsApp, SMS, Email).
- **Loyalty Integration:** Tier management and rewards.
- **Audience Segmentation:** Visual segment cards.
